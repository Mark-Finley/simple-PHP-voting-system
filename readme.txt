# Voting Management System

## Free Download Source Code

### Prerequisites

1. **XAMPP**
2. **Text Editor / IDE**: Notepad++/Vscode
3. **Voting_Management_System**

### Steps to Download and Set Up

1. Download the zip file or download WinRAR.
2. Extract the file and copy the `Voting_Management_System` folder.
3. Paste the folder inside the root directory where you installed XAMPP (e.g., `C:/xampp/htdocs`, `D:/xampp/htdocs`, `E:/xampp/htdocs`).
4. Open PHPMyAdmin: [http://localhost/phpmyadmin](http://localhost/phpmyadmin)
5. Create a database with the name `voting_db`.
6. Import the `voting_db.sql` file (provided inside the zip package in the `SQL file` folder).
7. Run the script: [http://localhost/Voting-Management-System](http://localhost/Voting-Management-System)
